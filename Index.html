How to Use Crypto Vault

  1. Set Up Your Vault: Choose a strong PIN and enter your name to create your vault.
  2. Add Wallets: Add your cryptocurrency wallets.
    Note: For demonstration, private keys are encoded but not securely encrypted.
  3. Unlock to Access: Use your PIN to unlock the vault and access your wallets.
  4. Backup: Use the Export feature to save a backup of your wallets.
    Warning: All data is stored locally in your browser. If you clear your browser data or use a different browser/device, your wallets will be lost.
  5. Using on Ubuntu/Linux: You can use Crypto Vault on Ubuntu or any Linux system by simply opening the index.html file in your web browser (e.g., Firefox, Chrome). No additional installation is required.
